# 月黑风高✌

## 题目描述

在一个风黑月高的晚上，牛大截获了一条敌对村狗村的加密信息：

```
P6XNy4{4Wbh-PhvAA-oIJAGGAGJ5-NbhBI-dWIvy-ysAGJ5}
```

于是它去请教牛家族中的密码手牛二，经验丰富的牛二只能找到一个加密公式：

```
C = ( a × P + b ) mod m
```

牛二想了半天都不知道是啥，只能来寻求你的帮助，它们承诺只要你帮助它们解出来就请你吃它们屯最好吃的牛肉串。
